# JSON Messages:

## QUESTION:
```javascript
{ 
    id    : INT,
    desc  : STR,
    topic : STR,
    diff  : INT,
    tests : [STR]
}
```

## TEST:
```javascript
{ 
    id   : INT,
    desc : STR,
    rel  : INT,
    sub  : INT,
    ques : [{QUESTION}],
    pts  : [INT]
}
```


## ATTEMPT:
```javascript
{ 
    id       : INT,
    test     : {TEST},
    answers  : [STR],
    grades   : [INT],
    comment  : STR,
    feedback : [STR],
    remarks  : [STR]
}
```


## addQ
### request
```javascript
{
    type  : addQ,
    desc  : STR,
    topic : STR,
    cons  : [STR],
    diff  : INT,
    tests : [STR]
}
```

### reply
```javascript
{
    type  : addQ,
    error : STR,
    que   : {QUESTION}
}
```
